# FlappyBird

A Flappy Bird-style game built almost entirely by AI agents, with Matthew as game director.

- **Engine:** Godot 4 (.NET / C#)
- **Target:** Android, Google Play Store
- **Approach:** TDD, vertical slices, agent-driven, no human-typed gameplay code
